# fount: stuff to build

## analysis (/doctor or /xray)
Need a way to see if the script actually works without reading the whole thing every time.
- **dialogue balance**: show me who's talking the most. if a side character has 80% of the lines in a scene, i probably messed up.
- **pacing heatmap**: a visual map of action vs dialogue. if it's all dialogue for 10 pages, it's a play, not a movie. if it's all action, it's an imax doc. need to find the balance.
- **scene length**: keep scenes under 3 pages. the app should just tap me on the shoulder when i'm rambling.

## structure (/beatmode)
seeing the "bones" of the story.
- **index cards**: a grid of scene summaries. eventually, i want to just grab a card and move it to a different act without cutting and pasting a thousand lines of text.
- **act visualizer**: a progress bar that shows where the midpoint and act breaks should be hitting. helps me see if the second act is dragging or if i'm rushing the ending.

## the time machine (/undotree)
normal undo is trash because if you undo and then type something, you lose everything you just "undid." 
- **undotree**: keep every version of every change in a branching map (like git or vim). if i delete a whole scene and regret it three days later, i should be able to just warp back to that specific branch.
